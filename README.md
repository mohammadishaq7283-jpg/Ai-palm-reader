# AI Mystic - Global Palm & Tarot Reader

A mystical, AI-powered web application that provides Palm Reading (Chiromancy) via image analysis and Tarot Card readings with multi-language support.

## Setup

1.  **Clone the repository**
2.  **Install dependencies:**
    ```bash
    npm install
    ```
3.  **Set Environment Variables:**
    Rename `.env.example` to `.env` and add your OpenRouter API key.
    ```env
    OPENROUTER_API_KEY=your_key_here
    ```
4.  **Run Development Server:**
    ```bash
    npm run dev
    ```
